Пояснительная записка к проекту "Тетрис".

Автор: Мифтахова Ева

Идея проекта заключается в том, чтобы разобраться и сделать почти идентичную игру с нуля "Тетрис" с помощью PyGame.

В этом проекте используется tkinter, для запуска начального окна с кнопкой "Начать!". База данных нужна для того, чтобы подсчитывать количество проигрышей. В классе Tetromino реализованы: поворот фигуры(вверх), def draw - рисовка фигуры, def move_down - движение фигуры вниз по полю и ускорение движение фигуры, def move_side - направление движения фигуры(вправо и влево), также есть проверка на заполненность линий, очистка заполненных, обозначение границ поля и фигур, их цвет и тд., проверка на проигрыш, в консоль выводится сообщение о том, что игрок проиграл.

Для запуска нужны библиотеки PyGame, random, tkinter(*), Модуль sys и база данных sqlite3.

Все интересующие скриншоты и видео  представлены в презентации.
